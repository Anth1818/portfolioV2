Sobre este proyecto

SIGESTEI es una aplicación web construida con Next.js para centralizar la gestión operativa y administrativa de recursos tecnológicos dentro de una institución o empresa.

Funcionalidades principales

  • Inventario de equipos: control completo de computadoras, periféricos y otros activos tecnológicos.
  • Gestión de solicitudes: registro y seguimiento de solicitudes de mantenimiento, soporte y asignación.
  • Administración de usuarios: gestión de usuarios con distintos roles y permisos.
  • Sistema de auditoría: trazabilidad de operaciones y cambios en tiempo real.
  • Dashboard analítico: visualización de métricas clave para apoyar la toma de decisiones.
  • Generación de reportes: exportación de información en PDF para control interno y seguimiento.

Impacto en la institución

Tras su implementación, SIGESTEI permitió mejorar la organización de procesos internos y reducir tareas manuales. En la operación diaria, se observaron impactos realistas como:

  • Reducción del tiempo de respuesta a solicitudes en aproximadamente 30%.
  • Menor cantidad de inconsistencias en inventario gracias a registros centralizados y auditables.
  • Mayor visibilidad para coordinadores y responsables mediante tableros con indicadores actualizados.
  • Mejor control de responsabilidades por usuario, facilitando auditorías y el seguimiento de incidencias.
